# FerroCopy**FerroCopy** は、TeraCopy にインスパイアされた Windows 向け高速ファイルコピーツールです。Rust で書かれており、並列コピー、ハッシュ検証、egui GUI、拡張シェル統合を備えています。## CLI 使用例```bash# ファイルコピーferrocopy source.txt dest_folder/# ファイル移動（成功時に元ファイル削除）ferrocopy --move source.txt dest_folder/# ハッシュ検証付きコピーferrocopy --verify blake3 source.iso dest.iso# GUI 起動ferrocopy --gui# シェル統合のインストール / アンインストールferrocopy --install-contextferrocopy --uninstall-context# ── 拡張機能 (v0.2.0) ──# SSH/SFTP リモートアップロードferrocopy --ssh user@host:22:/remote/path source.txt# SSH 秘密鍵指定ferrocopy --ssh user@host:/remote/path --ssh-key ~/.ssh/id_ed25519 source.txt# JSON-RPC サーバー起動 (ポート 9876)ferrocopy --rpc-listen 9876# ディレクトリ監視（新規ファイル自動コピー）ferrocopy --watch-dir ./incoming dest_folder/# Lua フィルタースクリプトferrocopy --lua-filter filter.lua source.txt dest/# WASM プラグイン実行ferrocopy --wasm-plugin my_plugin.wasm source.txt dest/# パイプライン: Scan → Copy → Verify → Reportferrocopy --use-schedule --verify blake3 source.txt dest/# I/O プールスレッド数指定ferrocopy --io-pool-threads 8 source.txt dest/```## 機能一覧| カテゴリ | 機能 | 状態 ||----------|------|:----:|| コア | 並列コピーエンジン（バックプレッシャー制御） | ✅ || コア | アダプティブバッファ（ファイルサイズ自動調整） | ✅ || コア | 指数バックオフ +
